# ludo-strategy-india

English educational content project for Ludo learners in India.

## Focus

- Practical Ludo strategy explained in calm, beginner-friendly English.
- Realistic board reading, decision-making, scenario study, and review habits.
- Teaching voice based on player observation, not promotion or hype.

## Writing Rules

- Follow `teenpatti-guide-india/内容模板.md`.
- Keep every article useful, readable, and internally consistent.
- Prefer real game reasoning: why a move matters, where players misread the board, and how to improve the next decision.
- Avoid gambling framing, marketing tone, fake authority, and keyword stuffing.

## Content Structure

- One clear topic per page.
- Exactly one hero image reference per article using an existing file from `assets/images`.
- Strong heading hierarchy, short-to-medium paragraphs, FAQ, summary, keywords, and related pages.
